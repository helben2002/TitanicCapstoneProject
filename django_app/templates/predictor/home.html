Titanic++ Survival Predictor

This project combines machine learning with a Django web application to predict the probability of survival for Titanic passengers based on historical data and engineered features.

Project Purpose

The goal of the Titanic++ project is to demonstrate how a machine learning pipeline can be integrated into a production-style web system. The project includes:

  • Data preprocessing and feature engineering
  • Model training and evaluation
  • A prediction Django web interface for predictions
  • A database storing historical predictions

System Design

The application consists of three main components:

Machine Learning Pipeline

Handles preprocessing, feature engineering, model training, and saving the final model.

Django Backend

Accepts user input, maps it to the model schema, runs predictions, and stores results in the database.

User Interface

Provides pages for predictions and viewing historical prediction data.
